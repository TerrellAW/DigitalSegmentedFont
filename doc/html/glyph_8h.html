int compute_advance ( Glyph *  glyph)

Computes the width between glyphs based on the glyph's segments.

Calculates the right-most edge of all segments and returns the highest value with a padding of 1 pixel.

Parameters
glyphPointer to the glyph whose advance width is being computed.
Returns
The computed advance width, or -1 if glyph is NULL.

Adds a 3x3 segment to the glyph at the specified grid position.

Automatically converts grid coordinates to pixel coordinates with 1-pixel gaps. Each grid position is 4 units apart (3-unit segment + 1-unit gap).

Parameters
glyphPointer to the glyph to modify.
grid_xGrid column position. Converted to pixel_x.
grid_yGrid row position. Converted to pixel_y.
Returns
0 on success, -1 if the glyph is NULL or the segment addition fails.

Adds a square segment to the glyph.

Dynamically resizes the segment array and appends a new segment of the specified size at the specified (x,y) coordinates.

Parameters
glyphPointer to the glyph to modify.
xX-coordinate of the new segment. Must be a valid integer.
yY-coordinate of the new segment. Must be a valid integer.
sizeWidth/Height of the square segment. Must be a valid integer.
Returns
0 on success, -1 if glyph is NULL or size is 0 or less.
